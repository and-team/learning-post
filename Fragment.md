# Fragment
Fragment 相关的东西

3.0 引入的，翻译为碎片或者片段。为了解决不同分辨率的动态和灵活 UI 设计。

* 如果 3.0 之前的系统，需要导入 android-support-v4
* Fragment 必须嵌入在 Activity 中使用
