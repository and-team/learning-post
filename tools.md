# tools

有些效果则必须在运行之后才能看见，比如这种情况：

> TextView在xml中没有设置任何字符，而是在activity中设置了text。因此为了在ide中预览效果，你必须在xml中为TextView控件设置android:text属性

## xmlns:tools

```xml
xmlns:tools="http://schemas.android.com/tools"
```
