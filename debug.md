# Android Debug 

## DDMS

* Dalvik Debug Monitor Service
* Android 开发环境中的 Dalvik 虚拟机调试监控服务

#### 提供：

* 为测试设备截屏
* 接收 SMS
* 模拟电话呼叫

#### 存放在：

Library/Android/sdk/tools/ddms
