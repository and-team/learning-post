# Java 基础 

## final

#### final 变量（常量）

```java
public final static String LOGIN_NAME = "loginname";
```


## static

#### static 变量

* 被 static 修饰的变量 - 静态变量或者类变量
* 没有被 static 修饰的变量 - 实例变量
