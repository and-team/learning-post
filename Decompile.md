# Android 反编译

apk 文件解压后由 2 部分的文件：

1、xml 文件
2、dex 文件，这里面有 class


## 步骤

1、将 APK 文件后缀从 .apk 改成 .zip，然后将 .zip 解压
2、在解压出来的目录中你可以看到 classes.dex
